Difference between JPA, Hibernate and Spring Data JPA

JPA - A set of rules/guidelines (specification) for ORM in Java

Hibernate - A tool that implements those JPA rules

Spring Data JPA  - A helper layer on top of Hibernate to reduce code

Analogy: JPA is like a SYLLABUS, Hibernate is the TEXTBOOK written based on that syllabus, Spring Data JPA is the SUMMARY NOTES that make it easier to study.
